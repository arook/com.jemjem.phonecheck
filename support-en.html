Frequently Asked Questions (FAQ)

1. What are the requirements to run the PhoneCheck iOS app?

• iOS 15.6 or higher
• iPhone device (iPad not supported)
• Same Wi-Fi network as the PhoneCheck desktop application
• Required permissions for various hardware tests (camera, microphone, etc.)

2. How do I connect to the PhoneCheck desktop application?

1. Ensure phone and computer are on the same Wi-Fi network
2. Enter the computer's IP address in the iOS app (displayed by desktop app)
3. Click the "Connect" button
4. Once connected, the desktop app can send test commands

3. Where is test data stored?

All diagnostic data is temporarily stored locally on the device and can be deleted after testing. Data can be optionally sent to the connected desktop application for further analysis. We do not store any diagnostic data on our servers.

4. What permissions does the app need and why?

The app requires the following permissions for hardware testing:
• Camera: Test front and rear camera functionality
• Microphone: Test audio input functionality
• Bluetooth: Test Bluetooth hardware
• Location: Test GPS functionality
• Face ID/Touch ID: Test biometric functionality
• Local Network: Discover and connect to desktop application

5. What to do if connection fails?

1. Check if phone and computer are on the same Wi-Fi network
2. Confirm firewall is not blocking port 8765
3. Try restarting the app and desktop software
4. Verify the computer IP address is correct
5. Ensure Bonjour service is working properly

6. What test types does the app support?

The app supports comprehensive hardware testing:
• Sensors (accelerometer, gyroscope, magnetometer, etc.)
• Connectivity (Wi-Fi, Bluetooth, GPS, NFC, etc.)
• Audio system (speaker, microphone, receiver)
• Cameras (front/rear camera, flashlight)
• Display and touch (screen pixels, multi-touch)
• Biometrics (Face ID, Touch ID)
• Battery and storage health status
